# 十文字学園女子大学・中岡加奈絵先生に関する調査報告書

## 1. 概要

中岡加奈絵（なかおか かなえ）先生は、十文字学園女子大学人間生活学部食物栄養学科に所属する准教授であり、子どもの食生活・食育および応用栄養学を専門とする研究者・教育者である[cite:39][cite:14]。幼児から中学生までの成長期を主な対象として、体格・骨量・体力・生活習慣・食習慣などを多面的に把握し、ライフステージに応じた栄養・食育の在り方を検討している点に特色がある[cite:28][cite:29][cite:31][cite:43]。また、ロコモティブシンドローム予防など、高齢化社会においても重要なテーマに取り組んでおり、日本家政学会奨励賞の受賞など、研究面での高い評価も得ている[cite:33][cite:34]。

## 2. 専門分野・研究テーマ

### 2-1. 専門分野

十文字学園女子大学の教員紹介および学部サイトによると、中岡先生の専門分野は「子どもの食生活・食育に関する研究」「応用栄養学」である[cite:14][cite:39]。研究者情報データベース（KAKEN）では、家政学および生活科学分野に属する研究者として登録されており、生活者の視点から健康と栄養を捉える家政学系のアプローチをとっていることがわかる[cite:34]。

### 2-2. 主な研究テーマ

公開されている論文や実践報告から、次のような研究テーマが確認できる。

- 小学校高学年児童における体型認識のずれ・体型不満と、身体状況・食習慣・運動習慣・体力との関連を検討する研究[cite:28]。
- 小学校5・6年生の体力と骨量、遊び、食品摂取状況との関連を検討し、成長期の体づくりと生活習慣の関係を明らかにする研究[cite:29]。
- 小学校高学年児童における社会的ジェットラグ（平日と休日の睡眠・起床時間のずれ）と食習慣・運動習慣との関連を分析する研究[cite:27]。
- 小学校高学年児童の家族との共食頻度や食事中の自発的コミュニケーションと、意思決定スキル・目標設定スキル・QOLとの関連を検討する研究[cite:25]。
- 中学生における食習慣・生活習慣および骨量の学年比較を行い、思春期の骨健康と生活習慣の関係を検討する研究[cite:31]。
- 東日本大震災後の東北三県の保育施設における給食の放射性物質対策、および食育の計画・実施・評価の現状を調査した研究[cite:26][cite:30]。

これらの研究から、中岡先生は「成長期（幼児・児童・生徒）の骨と体力」「生活リズム・食習慣・家族との共食」「震災後の保育施設における給食・食育」などを通じて、子どもの心身の健康と生活環境を総合的にとらえる研究を展開していると整理できる[cite:25][cite:28][cite:29][cite:30][cite:31]。

## 3. 研究業績（論文・著書など）

### 3-1. 学術論文・実践報告

- 『食育学雑誌』において、小学校5・6年生の体型認識のずれや体型不満と、身体状況・食習慣・運動習慣・体力の関連を検討した論文を発表し、ボディイメージと健康行動の関係に焦点を当てている[cite:28]。
- 同誌において、小学校5・6年生の体力と骨量・遊び・食品摂取状況の関連を扱った論文を発表し、骨量や体力と日常の遊び・食生活の関連を数量的に示している[cite:29]。
- 東北三県の保育施設における給食の放射性物質対策および食育の計画・実施・評価の現状について、震災後の長期的な視点から調査した論文を公表し、保育現場の安全対策と食育の両立に関する課題を明らかにしている[cite:30]。
- 『日本家政学会誌』では、中学生の食習慣・生活習慣・骨量の学年比較研究を通じて、成長段階の違いが生活習慣と骨健康に与える影響を検討している[cite:31]。
- 大学紀要には、埼玉県内の保育所園児（3～5歳児）とその保護者を対象とした実践・調査報告を掲載しており、大学倫理審査を経たフィールド調査に基づくエビデンスを蓄積している[cite:36]。

### 3-2. 著書・教科書

- 『応用栄養学実習―ケーススタディーで学ぶ栄養マネジメント』（健帛社）の共著者の一人として、ケーススタディ形式で応用栄養学を学ぶテキストの執筆に参画している[cite:38][cite:40][cite:44]。
- 『ライフステージ栄養学』（日本人の食事摂取基準2020年版準拠）の共著者として、各ライフステージに応じた栄養のポイントを整理した教科書の執筆に参加している[cite:37][cite:41]。
- 東京教学社から刊行されている『イラスト 応用栄養学』など、受験・学習参考書的な色彩の強い教材の著者としても名前が挙がっており、学生・受験生にとってのわかりやすい学習ツールの開発にも関わっている[cite:45]。

これらの著書・教科書・教材は、管理栄養士養成課程の学生だけでなく、実務者・受験生にとっても利用されており、中岡先生の「教育・実務に役立つ応用栄養学」に対する志向を反映していると考えられる[cite:37][cite:38][cite:40][cite:45]。

## 4. 受賞・評価

十文字学園女子大学のニュースによると、中岡先生は日本家政学会第77回大会（2025年、横浜国立大学）において、ロコモティブシンドローム予防に関する研究で日本家政学会奨励賞を受賞している[cite:33]。この研究は、高齢化社会における健康寿命の延伸と生活の質向上に資するものとして評価されており、子どもを主対象とする日常の研究に加えて、ライフコース全体を視野に入れた健康・栄養研究にも取り組んでいることがうかがえる[cite:33][cite:34]。

## 5. 教育活動・担当授業

### 5-1. 担当授業

十文字学園女子大学の授業情報によれば、中岡先生は以下のような科目を担当している[cite:43]。

- 子どもの食と栄養
- 応用栄養学Ⅰ（1Aクラス、1Bクラス）
- 応用栄養学実習（1Cクラス）
- 応用栄養学（上級学年向け科目） など

これらの科目では、子どもの発育・発達と栄養、ライフステージごとの栄養管理、ケーススタディを用いた応用栄養マネジメントなどを学ぶ内容が中心であり、講義と演習・実習を通じて「考えて実践できる管理栄養士」の育成を目指していると考えられる[cite:37][cite:38][cite:40][cite:43]。

### 5-2. ゼミ活動

大学のプレスリリースによると、2026年3月に実施された「砂糖のひみつを科学で解き明かす！」ワークショップにおいて、中岡ゼミ（指導教員：中岡加奈絵先生）が企画・運営を担当している[cite:42]。このワークショップは、地域社会や企業と連携した学外活動の一環として位置づけられ、砂糖の科学的性質や身体への影響をわかりやすく伝える内容となっている[cite:42]。資料によれば、中岡ゼミを含む各研究室が地域社会・企業と連携したアウトリーチ活動を積極的に展開していることが強調されており、ゼミ活動が「学外に開かれた実践の場」として機能していることが分かる[cite:42]。

また、学部サイトの「授業・研究室紹介」において、中岡先生の研究キーワードとして「子どもの食生活・食育」「応用栄養学」が掲げられており、ゼミ活動においても、子どもを対象とした食育プログラムや、生活習慣・体力・骨量に関する調査・実践が行われていると推察される[cite:14][cite:36][cite:43]。

## 6. 地域活動・社会貢献

中岡先生の地域活動・社会貢献として、以下のような点が挙げられる。

- 埼玉県内の保育所園児と保護者を対象とした実践・調査研究を通じて、地域の保育現場と連携しながら食育・健康支援に取り組んでいる[cite:36]。
- 東北三県の保育施設における震災後の給食・食育の実態調査は、被災地域の子どもと家庭を支えるための実践的課題を明らかにするものであり、公衆栄養・食育の観点から地域再建を支援する研究と位置づけられる[cite:26][cite:30]。
- 「砂糖のひみつを科学で解き明かす！」ワークショップのように、地域社会や企業と連携して食と栄養に関する科学的知見をわかりやすく伝えるアウトリーチ活動を行っている[cite:42]。

これらから、中岡先生は「子ども」「保育・教育現場」「地域社会」を結びつける実践的な研究・活動を重視していることがうかがえる[cite:26][cite:30][cite:36][cite:42]。

## 7. 中岡先生の思想・行動様式の特徴（公開情報からの整理）

公開情報から読み取れる範囲で、中岡先生の思想・行動様式を研究室ホームページ向けに整理すると、次のような特徴が挙げられる。

1. **子どもの成長を「からだ」と「こころ」と「生活」の全体としてとらえる視点**  
   - 骨量・体力・体型認識・QOL・意思決定スキルなど、多様な指標を用いて子どもの状態を把握している点から、単に栄養素摂取にとどまらず、子どもの生活世界全体を視野に入れた研究姿勢がうかがえる[cite:25][cite:28][cite:29][cite:31]。

2. **生活リズムや共食、遊びといった「日常生活」の重要性を重視**  
   - 社会的ジェットラグ、家族との共食、自発的コミュニケーション、遊びの種類など、生活リズムや家庭・友人との関わり方に焦点を当てていることから、「日々の暮らしの中の小さな習慣」が子どもの健康・発達に大きく影響するという考え方を重視していると整理できる[cite:25][cite:27][cite:29]。

3. **フィールドに根ざした実証的アプローチ**  
   - 保育所・小学校・中学校・保育施設など、実際の教育・保育現場での調査・実践を積み重ね、倫理審査を経た研究として成果をまとめている点から、現場と共同しながら課題を抽出し解決策を探る実証的スタイルが特徴的である[cite:26][cite:30][cite:36]。

4. **教育・実務に還元可能な教材・教科書づくりへの関与**  
   - 応用栄養学実習のテキストやライフステージ栄養学、イラスト応用栄養学などの教科書・学習書の執筆に関わっていることから、「学びやすさ」「実務で使える知識」に配慮した教材づくりに力を入れていると考えられる[cite:37][cite:38][cite:40][cite:45]。

5. **ライフコース全体を見通した健康・栄養観**  
   - ロコモティブシンドローム予防に関する研究で家政学会奨励賞を受賞していることから、子どもの時期に加え、成人・高齢期も視野に入れた生涯的な健康・栄養観を持っていると解釈できる[cite:33][cite:34]。
