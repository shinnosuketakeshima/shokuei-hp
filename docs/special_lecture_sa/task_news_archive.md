# タスク：新着情報一覧（アーカイブ）ページの作成

新着情報の「一覧へ」リンクの遷移先となる、全てのニュースを確認できるページを作成する。

## ステップ

1. **コンポーネント作成**
    - [ ] `src/components/NewsArchive.jsx` を新規作成
        - `News.jsx` と同様にFirestoreとDUMMY_NEWSからデータを取得
        - 全てのニュースをリスト表示
        - カテゴリー（全て、教員コラム、学生コラム、イベント、お知らせ）によるフィルタリング機能を追加
        - デザインを「コラム一覧」として見やすいレイアウトにする（サムネイル画像があれば尚良いが、現状はテキスト主体）

2. **ルーティングとリンクの設定**
    - [ ] `src/App.jsx` に `/news` ルートを追加
    - [ ] `src/components/News.jsx` の「一覧へ」のリンク先を `/news` に変更

3. **ヘッダーの修正（必要に応じて）**
    - [ ] ナビゲーションに「ニュース」や「コラム」への直接リンクを追加するか検討

4. **仕上げと確認**
    - [ ] 各カテゴリーでのフィルタリングが正しく動作するか確認
    - [ ] モバイル表示でのレイアウト確認
    - [ ] デプロイ
