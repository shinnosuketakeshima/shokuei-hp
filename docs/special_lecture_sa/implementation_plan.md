# 実装計画：神山先生の教員コラム（総合科目SAレポート）

## 1. 概要
神山先生が担当する総合科目「いちから学ぶnutrition science」における、神山ゼミ生によるSA（Student Assistant）活動のレポートをコラムとして公開します。

## 2. 実装詳細

### A. 新規コンポーネント作成
- ファイル名: `src/components/LabKamiyamaReport.jsx`
- 内容:
    - ヒーローセクション（タイトル、日付、著者：神山真澄先生）
    - 本文セクション（抽出したテキストを使用）
    - 画像セクション（抽出した2枚の画像を使用）
    - SA学生のコメントセクション（引用スタイルで強調）
    - 神山研究室へのリンク（フッター付近）

### B. ルーティング設定
- ファイル: `src/App.jsx`
- パス: `/lab-kamiyama-report`
- メタデータ:
    - タイトル: 「いちから学ぶnutrition science」SAレポート | 食物栄養学科
    - 説明: 総合科目「いちから学ぶnutrition science」での神山ゼミ生によるSA活動レポート。アクティブラーニング形式の授業の様子や学生の成長を伝えます。

### C. 新着情報追加
- ファイル: `src/components/News.jsx`
- 項目:
    - 日付: 2026.5.15 (本日)
    - タグ: 教員コラム
    - タイトル: 先輩が授業をサポート！総合科目「いちから学ぶnutrition science」SAレポート
    - リンク: `/lab-kamiyama-report`

## 3. デザイン方針
- 清潔感があり、教育の現場が伝わる温かみのあるデザイン。
- マヨネーズ作りや工作の楽しさが伝わるよう、画像を大きく配置。
- 学生のコメントは、背景色を変えるなどして、先生の文章と区別しやすくする。
